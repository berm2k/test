# Images Repository

This directory is for storing image files that you can reference with direct links.

## How to Use

1. Upload your image files to this `images/` directory
2. Reference them using the raw GitHub URL format:
   ```
   https://raw.githubusercontent.com/berm2k/test/main/images/[filename]
   ```

3. Since this is a private repository, only authorized users can access these links

## Example

If you upload `logo.png` to this folder, the link would be:
```
https://raw.githubusercontent.com/berm2k/test/main/images/logo.png
```

You can embed this in Markdown, HTML, or use it anywhere you need the image URL.
